## 🎯 Sobre o projeto

Este repositório contém um sistema simples de validação utilizando Expressões Regulares (Regex), desenvolvido com HTML, CSS e JavaScript puro.

O projeto permite verificar se um valor inserido pelo usuário é um número válido, utilizando uma expressão regular para garantir que apenas dígitos sejam aceitos.

O objetivo deste projeto foi demonstrar o uso prático de Regex para validação de dados em formulários, criando uma aplicação leve, interativa e educativa.

## ✍️ Como personalizar rapidamente

Alterar validação: edite a expressão regular no JavaScript.

Exemplo:

const regex = /^\d+$/;

Permitir letras: modifique a regex para incluir outros padrões.

Mudar cores: altere as propriedades style.color no JavaScript ou no CSS.

Customizar layout: edite o CSS da classe .container.
