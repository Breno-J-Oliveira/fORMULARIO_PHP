## 📖 Sobre

Sistema CRUD completo para cadastro de bolos com **PHP + MySQL**. Funcionalidades: cadastro com upload de imagem, listagem em tabela estilizada, edição e exclusão. Interface em tons pastéis. O `index.php` (137 linhas) concentra toda a lógica (rotas, banco, formulário, listagem).

## 🛠️ Tecnologias

- **PHP** — Lógica de servidor e templates dinâmicos
- **MySQL** — Banco de dados relacional (quando aplicável)
- **HTML/CSS** — Interface e estilização
