## O que é esta categoria?

A categoria **Estudos de PHP** é o maior bloco de aprendizagem do repositório — **15 projectos** que cobrem a jornada completa do desenvolvimento web back-end com PHP. Da primeira atividade com variáveis e formulários GET/POST até sistemas CRUD completos com MySQL, passando por autenticação de utilizadores, upload de ficheiros, validação com Regex e interfaces responsivas com Bootstrap.

É aqui que o back-end ganha vida: dados que entram por formulários, são validados, sanitizados, persistidos em base de dados e devolvidos ao utilizador em interfaces dinâmicas. A stack LAMP clássica aplicada em projectos reais.

**Do primeiro `echo` ao sistema de biblioteca com MySQL.**

---

## O que esta categoria entrega?

### Para o estudante de PHP
- **Sintaxe PHP** — variáveis, arrays associativos, condicionais, laços, funções
- **Formulários e superglobais** — `$_GET`, `$_POST`, `$_SESSION`, validação, sanitização
- **CRUD com MySQL** — `INSERT`, `SELECT`, `UPDATE`, `DELETE` com queries parametrizadas
- **Upload de ficheiros** — `$_FILES`, validação de tipo/tamanho, armazenamento
- **Autenticação** — sistema de login com sessões, controlo de tentativas
- **Bootstrap** — interfaces responsivas, grid system, componentes
- **Expressões regulares** — validação avançada de campos (email, telefone, CPF)
- **Arquitectura procedural** — separação de responsabilidades (conexão, lógica, apresentação)
- **Design responsivo** — media queries, flexbox, grid, mobile-first

### Para o recrutador / avaliador
- **15 projectos documentados** — do iniciante ao intermediário
- **4 sistemas CRUD completos** — Biblioteca, Locadora, Bolos, Jogos
- **Domínio de MySQL** — modelagem, queries, relações entre tabelas
- **Stack LAMP** — PHP + MySQL + Apache, a base de milhares de sistemas em produção
- **Validação e segurança** — Regex, sanitização, queries parametrizadas contra SQL Injection
- **Interfaces profissionais** — Bootstrap, design responsivo, dark mode
- **Projectos interativos** — quiz com sessões, pontuação, múltiplas telas

## Como funciona? — Percurso de Aprendizagem

### 🔰 Fase 1 — Fundamentos (Projectos 1–3)

```
1. comecodophp — 5 atividades introdutórias
   Variáveis, arrays, condicionais, laços
   Primeiros formulários GET/POST
       ↓
2. aaaaaaaaaa — Cards de Yu-Gi-Oh!
   Arrays associativos, $_GET
   CSS flexbox para layout de cards
       ↓
3. Atividade 05.09.2025 — 5 projetos web responsivos
   Cardápio Digital, Galeria de Imagens,
   Landing Page de Evento, Blog Simples,
   Página de Apresentação Pessoal
   Media queries, flexbox, grid, mobile-first
```

**Objectivo:** dominar a sintaxe PHP, perceber o ciclo request/response e criar interfaces web responsivas.

### 🟡 Fase 2 — Formulários e Validação (Projectos 4–8)

```
4. Formulário de Ingresso em PHP
   Validação de dados, sanitização
       ↓
5. Formulário de Interação
   Campos dinâmicos, processamento no back-end
       ↓
6. Formulário Personalizado
   Layout customizado, estilização avançada
       ↓
7. Tabela Dinâmica
   Geração de tabelas HTML com PHP
       ↓
8. Validação com Regex
   Expressões regulares para email, telefone, CPF
```

**Objectivo:** processar dados de formulários com segurança — validar, sanitizar e responder ao utilizador. Dominar Regex para validação avançada.

### 🟢 Fase 3 — CRUD com MySQL (Projectos 9–12)

```
9. Formulário PHP — CRUD de Cadastro de Bolos
   PHP + MySQL + Upload de Imagens
       ↓
10. Registro de Jogos Jogados
    CRUD completo (create, read, update, delete)
    Conexão PDO, queries parametrizadas
       ↓
11. Sistema de Biblioteca
    PHP + MySQL (livros, autores, empréstimos)
    Modelagem relacional
       ↓
12. Locadora de Eletrônicos
    PHP + Bootstrap + MySQL
    Interface profissional com componentes Bootstrap
```

**Objectivo:** construir sistemas CRUD completos com base de dados relacional. Dominar o ciclo completo: formulário → validação → base de dados → resposta.

### 🔵 Fase 4 — Autenticação e Projectos Interativos (Projectos 13–15)

```
13. Sistema de Login Básico
    Front-end com validação, 3 tentativas, dark mode
       ↓
14. Quiz do Ben 10
    PHP procedural, $_SESSION, 3 telas, pontuação
       ↓
15. Compensação de Aula (05/06)
    Exercícios práticos consolidados
```

**Objectivo:** implementar autenticação com sessões e criar aplicações interativas com estado (quiz com pontuação).

## Como Usar Esta Categoria para Estudar

### Pré-requisitos
- **XAMPP**, **WAMP** ou **Laragon** (Apache + PHP + MySQL)
- Navegador web
- Editor de código (VS Code recomendado)
- MySQL Workbench ou phpMyAdmin (para gerir bases de dados)

### Passo a passo

1. **Instala o XAMPP** — liga os serviços Apache e MySQL
2. **Cola a pasta `_estudos-php` dentro de `htdocs`** (XAMPP) ou `www` (WAMP)
3. **Começa pelo `comecodophp`** — 5 atividades progressivas do básico ao formulário
4. **Explora os formulários** — percebe o fluxo `$_GET`/`$_POST` → validação → resposta
5. **Estuda os CRUDs** — importa os ficheiros `.sql` no MySQL e analisa cada operação
6. **Testa os sistemas** — acede via `http://localhost/_estudos-php/Registro-de-Jogos-Jogados/`
7. **Constrói o teu próprio CRUD** — escolhe um domínio (receitas, contactos, filmes) e implementa

### Estrutura de cada projecto

Cada subpasta contém:
- **`README.md`** — objectivos, conceitos, instruções
- **Ficheiros `.php`** — lógica de servidor
- **Ficheiros `.html`**, `.css`, `.js` — front-end
- **Ficheiros `.sql`** ou `.txt` — esquemas de base de dados
