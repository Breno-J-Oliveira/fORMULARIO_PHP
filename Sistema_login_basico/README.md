## 🎯 Sobre o projeto

Este repositório contém um sistema simples de login chamado Cyber Security Senai — uma interface de autenticação desenvolvida com HTML, CSS e JavaScript puro.

O projeto simula um ambiente de segurança digital, permitindo validar usuários através de um sistema de login com credenciais pré-definidas e controle de tentativas de acesso.

O objetivo deste projeto foi criar uma aplicação leve, funcional e visualmente coerente com o tema de cibersegurança, focando em lógica de autenticação no front-end.

## 🔍 Demonstração / Preview

Coloque o arquivo em um navegador para ver:

Tela de login centralizada

Campos de usuário e senha

Botão de autenticação

Sistema de validação com mensagens de alerta

Controle de tentativas (máximo de 3)

(Substitua os usuários no código para testar diferentes acessos.)

---

## ⚙️ Funcionalidades principais

Sistema de login com usuários simulados (mockados).

Controle de tentativas com bloqueio após 3 erros.

Mensagens de feedback dinâmicas (sucesso/erro).

Interface minimalista com tema escuro.

Efeitos de hover em título e botão.

## ▶️ Como rodar localmente

Crie um arquivo:

index.html

Cole o código do projeto dentro dele.

Abra no navegador (ou utilize um servidor local, ex.: npx http-server).

Dica: usar servidor local pode evitar problemas em projetos maiores.

---

## ✍️ Como personalizar rapidamente

Alterar usuários: edite o array users no JavaScript.

Ajustar tentativas: modifique a variável attempts no script.

Mudar cores: altere os valores no CSS (background-color, color, etc).

Melhorias: implemente backend, criptografia ou banco de dados.
